|------------|----------|---------|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------|

| voice      | YES      |         | multiple and depending of the language                                             | Check available names on the web site                                       |
| cache      | No       | TRUE    | True / False                                                                       | True if you want to use the cache with this TTS                             |
